# 📱 Настройка проекта (с телефона, без компьютера)

Эта инструкция написана для владельца ВКУСНОМАРКЕТа без опыта программирования.
Все шаги делаются с Android-телефона через браузер.

> **Главное правило:** ничего страшного, если что-то не получится с первого раза. Любой шаг можно повторить. Никакие данные не теряются.

---

## ✅ Шаг 1. Деплой на Vercel (показать сайт интернету)

Vercel — это бесплатный сервис, который сам собирает приложение из кода на GitHub и публикует его в интернете. Привязка занимает 5 минут.

1. Откройте [https://vercel.com/signup](https://vercel.com/signup) на телефоне.
2. Нажмите **«Continue with GitHub»** → войдите своим аккаунтом GitHub.
3. После входа: **«Add New… → Project»**.
4. Найдите ваш репозиторий **`vkusnomarket05`** → нажмите **«Import»**.
5. На экране настроек ничего не меняйте → нажмите **«Deploy»**.
6. Через 1–2 минуты Vercel выдаст ссылку вида `https://vkusnomarket05-xxx.vercel.app` — это и есть ваше приложение.

После каждого `git push` в репозиторий Vercel будет автоматически пересобирать сайт.

## ✅ Шаг 2. Telegram-бот (уведомления о заказах)

Чтобы каждый новый заказ приходил вам в Telegram:

### 2.1 Создать бота
1. В Telegram откройте чат с **@BotFather**.
2. Отправьте команду: `/newbot`
3. Имя бота: **ВКУСНОМАРКЕТ Заказы** (или любое).
4. Username: должен заканчиваться на `bot`, например **`vkusnomarket_orders_bot`**.
5. BotFather пришлёт строку вида:
   ```
   1234567890:AAFmaB-xxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   Это **TELEGRAM_BOT_TOKEN**. Скопируйте.

### 2.2 Узнать ваш chat_id
1. Найдите своего нового бота в Telegram (по username).
2. Нажмите **«Start»** или просто напишите ему любое сообщение (например, «привет»).
3. Откройте в браузере (вместо `<TOKEN>` подставьте токен из шага 2.1):
   ```
   https://api.telegram.org/bot<TOKEN>/getUpdates
   ```
4. В ответе найдите `"chat":{"id": 123456789, ...}` — число `123456789` это **TELEGRAM_ADMIN_CHAT_ID**.

### 2.3 Добавить в Vercel
1. Vercel → ваш проект → **Settings → Environment Variables**.
2. Добавьте переменные:
   - `TELEGRAM_BOT_TOKEN` = ваш токен
   - `TELEGRAM_ADMIN_CHAT_ID` = ваш chat_id
3. Нажмите **Save** → затем **Deployments → ⋯ → Redeploy** последнего деплоя.

Готово! При каждом заказе вам в Telegram будет приходить сообщение:
```
🛒 Новый заказ № VKM-...
Клиент: Артур
Телефон: +7 928 ...
Адрес: ...
Состав: ...
Итого: 1240 ₽
```

## ✅ Шаг 3. Поменять админский пароль

По умолчанию: логин `admin`, пароль `vkusno2025` — обязательно поменяйте!

1. Vercel → Environment Variables.
2. Добавьте:
   - `ADMIN_LOGIN` = ваш логин (например, `boss`)
   - `ADMIN_PASSWORD` = ваш надёжный пароль
3. Redeploy.

Теперь вход в админку (`https://ваш-сайт/admin/login`) будет под новыми данными.

## ✅ Шаг 4. Заполнить контакты поддержки

Откройте файл `src/lib/constants.ts` (на GitHub можно редактировать прямо в браузере) и заполните:

```ts
export const DEFAULT_CONTACTS = {
  whatsapp: "+7 928 000 00 00",   // ваш WhatsApp
  telegram: "@vkusnomarket",       // ваш Telegram
  phone: "+7 928 000 00 00",       // телефон поддержки
  email: "info@vkusnomarket.ru",  // email (необязательно)
};
```

После сохранения Vercel автоматически пересоберёт сайт с новыми контактами.

## ✅ Шаг 5. SMS.ru (реальные SMS-коды для входа) — опционально

Сейчас вход в приложение работает в **демо-режиме**: код подтверждения всегда `1234`. Когда захотите реальные SMS:

1. Зарегистрируйтесь на [https://sms.ru/](https://sms.ru/) (для ИП).
2. В личном кабинете → **API → API_ID** — скопируйте ID.
3. Добавьте в Vercel:
   - `SMSRU_API_ID` = ваш api_id
   - `SMSRU_FROM` = имя отправителя (обычно `SMS_AERO` или `INFORM` пока не одобрено своё)

> Подключение SMS требует небольшого изменения кода — попросите помочь, когда будете готовы.

## ✅ Шаг 6. Supabase — постоянное хранение заказов

По умолчанию заказы и курьеры хранятся в памяти процесса Vercel и **сбрасываются** при холодных стартах серверной функции. Telegram-уведомление о заказе при этом приходит, но заказ исчезает из админки. Чтобы починить — подключите Supabase:

1. Зарегистрируйтесь на [https://supabase.com/](https://supabase.com/) → **Start your project** (войдите через GitHub).
2. **New Project** → название `vkusnomarket`, регион **Frankfurt**, придумайте пароль БД.
3. Подождите 1–2 минуты, пока проект создастся.
4. **Project Settings (⚙) → API** → скопируйте 3 значения:
   - **Project URL** → `NEXT_PUBLIC_SUPABASE_URL`
   - **anon public** → `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - **service_role secret** → `SUPABASE_SERVICE_ROLE_KEY` (никому не показывайте!)
5. В Vercel → ваш проект → **Settings → Environment Variables** добавьте все 3 (для всех окружений: Production, Preview, Development).
6. В Supabase → **SQL Editor** → **New query** → вставьте содержимое файла [`supabase/migrations/0001_initial_schema.sql`](./supabase/migrations/0001_initial_schema.sql) → нажмите **Run**.
7. В Vercel → **Deployments** → у самого верхнего «...» → **Redeploy** (без этого новые env-переменные не подхватятся).

После этого:
- Заказы перестанут пропадать.
- Курьеров можно добавлять/удалять через админку, и они тоже сохранятся.
- Статусы заказов меняются и видны клиенту.

Если переменные пустые — приложение работает в режиме in-memory (как раньше).

## ✅ Шаг 7. Яндекс.Карты — опционально

Чтобы показать карту с адресом в админке и в заказе клиента:

1. Зарегистрируйтесь на [https://developer.tech.yandex.ru/](https://developer.tech.yandex.ru/).
2. Создайте API-ключ для **JavaScript API + HTTP Геокодер**.
3. В Vercel добавьте `NEXT_PUBLIC_YANDEX_MAPS_API_KEY` = ваш ключ.

## ✅ Шаг 8. Установка приложения на телефон (PWA)

Когда сайт открыт в Chrome на Android:

1. Меню (⋮) → **«Установить приложение»** или **«Добавить на главный экран»**.
2. Иконка ВКУСНОМАРКЕТ появится на главном экране.
3. Открывается как обычное приложение, без адресной строки браузера.

Так же можно поделиться ссылкой с клиентами — они тоже установят к себе.

## ✅ Шаг 9. Покупка домена (необязательно)

Чтобы вместо `vkusnomarket05.vercel.app` был красивый адрес типа `vkusnomarket.ru`:

1. Купите домен на [reg.ru](https://reg.ru/), [beget.com](https://beget.com/) или у любого регистратора (~500 ₽/год).
2. В Vercel: ваш проект → **Domains → Add** → введите `vkusnomarket.ru`.
3. Vercel покажет DNS-настройки — скопируйте их в личном кабинете регистратора.
4. Через несколько часов домен заработает.

## 🆘 Если что-то сломалось

1. Зайдите в Vercel → **Deployments** → последний деплой со статусом «Error».
2. Откройте лог сборки — там будет описание ошибки.
3. Пришлите мне скриншот, я подскажу, что поправить.

Главное — не паникуйте. Откатиться к предыдущей рабочей версии в Vercel — это одна кнопка **«Promote to Production»** на любом старом деплое.
